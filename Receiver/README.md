
# Receiver Circuit Diagram and Components

The list of components of the receiver board is:
- Arduino uno r3
- 4 pin Photoresistor, LDR Module – Optical Sensitive Resistance Light Detection Sensor Module
- x3 LEDs: 2 green and one red
- x3 Buzzers
- LCD
- Rotary potentiometer
- Push button
- 270 Ω resistor
