
# Transmitter Circuit Diagram and Components

The list of components of the transmitter board is:
- Tiva C
- Ultrasonic Sensor
- Smoke Sensor (MQ2)
- Magnetic Sensor (Magnetic Reed Switch)
- A magnet
- Bluetooth module (HC-05)
- White LED
- USB to TTL for power source
